How to use this dashboard

Each row = one beat inside one scene of one story inside one vertical — exactly how the TGK 2.0 shell renders live demos.

  • Click any row to open the live vignette + persona + hotspot preview on the right. Edit the headline or lede — changes update the table in real time.
  • Funnel navigation filters by which taxonomy tier a beat primarily touches (Tier 2 = capabilities, Tier 4 = vertical data, etc.).
  • Audit flags map to the 5-layer taxonomy: cap (capability naming), brand (vertical tokens), arc (story arc / persona handoff), points (point-solution accuracy), parity (current vs future-state interface).
  • Persona handoff is green when the actor stays the same beat-to-beat; red when it switches without a pause gate.
  • Run Full Audit simulates scanning all beats against 12 named rules and adds realistic flags.
  • Apply Fix in the detail panel simulates writing back to the correct taxonomy layer (Beat → Story → Vertical).
